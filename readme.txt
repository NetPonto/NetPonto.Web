Website for Comunidade NetPonto (http://netponto.org)
=====================================================

To start hacking 
================

In powershell: '.\psake setup'


Optional
========

To build:
Powershell

To manage dependencies:
 - Ruby ( download installer from http://www.ruby-lang.org/en/downloads/ )
 - RubyGem at least 1.3.7 ( run 'gem update --system' after installing ruby to ensure latest version )
 - Nubular ( gem install nu )
